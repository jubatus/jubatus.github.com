How to Contribute to Jubatus¶

First of all please subsribe to the mailing list to reach developers’ and users’ community. Currently we don’t separate dev ML and user ML because Jubatus has just started now. Please,

  • Give us a feedback or tell us how do you use Jubatus in your environment via the mailing list .
  • Submit a bugreport to github issues .
  • Improve documentation or Suggest new feature by sending pull request .
  • Fix bugs .

Repositories¶

  • Jubatus
  • Jubatus software including classifier, recommender, stat, recommender and server framework.
  • branches - Committers use git-flow under A successful Git branching model .
  • develop - Development branch. Latest features are pushed here.
  • master - Release branch.
  • Website
  • Sphinx source of website (this site). Every contribution such as typo fix, English grammer fix will be welcomed.
