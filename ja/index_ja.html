Jubatus : オンライン機械学習向け分散処理フレームワーク¶

Jubatus は以下の特徴を持ったオンライン機械学習向け分散処理フレームワークです。

  • オンライン機械学習ライブラリ: 多値分類、線形回帰、推薦（近傍探索）、グラフマイニング
  • 特徴ベクトル変換器 (fv_converter): データの前処理と特徴抽出
  • フォルトトレラントな分散機械学習のためのフレームワーク

See Jubatus : Distributed Online Machine Learning Framework for English documentation.

Get Jubatus¶

Jubatus の最新バージョンは 0.3.4 です。Jubatus は LGPL v2.1 の下で配布されています。

  • ソースコードのダウンロード: TAR / ZIP / Git
  • バイナリパッケージ: Quick Start をご覧ください。
